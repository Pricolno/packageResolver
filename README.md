# packageResolver


```
git clone https://github.com/Pricolno/packageResolver.git

cd src 
```

## Usage

Get latest version packages

```
python3 packageResolver.py --latest_version NAME-PACKAGE
```

Get resolve package dependency



```
python3 packageResolver.py -td --version=NUMBER-VERSION  NAME-PACKAGE 
``` 


## Example 
```
python3 packageResolver.py --latest_version pidgizing
```

```
python3 packageResolver.py -td --version=0  digs ok
```
